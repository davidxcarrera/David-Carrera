Diseño de dispensador de platos

Diseñar un dispensador de platos funcional con todos los detalles y calculaciones correctas para que pueda cumplir con la velocidad y eficiencia requerida El objetivo de este proyecto fue familiarizarce con Solidworks y poder llevar un proyecto de diseño 3D a la realidad. Las calculaciones realizadas en este proyecto fue encontrar centro de masa y calcular la cantidad de fuerza requerida para separar los platos y gestionar un sistema de engranajes para cumplir los requisitos.

Diseñar un sistema anti hielo

Uno de los problemas de los aeroplanos debido al clima es el hielo que se puede producir en sus alas. En este proyecto, el objetivo es analizar el flujo y transferencia de calor en las alas, en este caso el sistema de anti hielo. Mediante el uso de ecuaciones de transferencia de calor y las simulaciones de del flujo de aire sobre las alas fue posible encontrar una solución viable.

Construción de Brazo Robotico

El diseño de este sistema mecatrónico se encarga de dirigir un espejo que hace comunicación con un satélite mediante un láser durante un espacio libre óptico. Con el uso de Solidworks se modeló el prototipo. Un controlador PID fue requerido para manipular el espejo óptico. El objetivo de este proyecto era diseñar y programar un brazo robótico que pueda ayudar a personas que sufren de DMD. En este caso la acción de este robot se limitan a tomar fotografías mediante el estímulo de músculos. El modelo del brazo robótico fue diseñado en SOLIDWORKS. Para manufacturar este prototipo hicimos uso de una cortadora láser. La programación del robot fue realizada mediante Python. La idea principal es estimular los músculos para enviar órdenes al robot.
